# openpost

This is an open post application, which is a simple integration of SparkJava and MongoDB.

Prerequisites
- install Java8
- SparkJava
- Eclipse IDE
- MongoDB
To compile:
- mvn clean package
- mvn eclipse:eclipse

To run:
- Run the PostApp main class
- Then the server is accessible at http://localhost:8080/
